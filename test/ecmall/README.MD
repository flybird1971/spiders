主要包含以下功能：

    1.1. 银行卡申请
    1.2. 银行卡密码重置
    1.3. 取款
    1.4. 存款
    1.5. 账号借贷流水记录查看
    1.6. 第三方支付接口   ： 每日消费限额

    2.1. 商城用户注册
    2.2. 商城用户密码重置
    2.3. 购物
    2.4. 查看购物车
    2.5. 查看之前购物
    2.6. 支付

    3.1. 商城后台  用户名：admin  密码 admin
    3.2. 商城后台配置商城银行卡号


*******************************************************************
功能流程：

    1.1 . 商城申请银行卡号 -》 登录商城后台 -》 配置商城银行卡号
    1.2.  登录商城后台，配置商城银行卡号
    1.3.  登录商城后台，添加商品

    2.1.  用户申请银行卡
    2.2.  用户登录银行系统，测试存钱，取款，重置密码等功能

    3.1.  用户到商城注册用户，测试重置密码功能
    3.2.  用户选购商品，查看购物车
    3.3.  用户网银支付
    3.4.  用户查看购物车商品状态是否切换为已支付

    4.1.  用户登录银行，查看金额，流水记录，并导出流水记录

    5.1.  商城管理者登录银行查看 账号余额，流水金额，并导出流水记录

    6.1.  用户多次购物，测试支付限额，流水记录。并导出流水记录


*******************************************************************
系统入口：

    1. 银行登录入口   bank.py
    2. 商城登录入口   shop.py
    3. 商城后台入口   admin.py
    4. 第三方支付接口 pay.py   暂时不实现，预留

数据存储

    暂时未学习mysql,先用文件读写处理数据【顺便练习文件读写): 】

    数据都被保存在./data/目录下，相关解释如下：

        bank_accout 银行用户金额账号记录
        bank_user   银行注册用户记录
        bank_config 银行系统相关配置
        flow.record 银行流水记录

        shop_goods   商城商品数据
        shop_user    商城用户数据
        shop_config  商城系统配置
        shop_records 商城日志记录
        shop_car     商城购物车记录

    后台用户，配置在shop_config文件中
